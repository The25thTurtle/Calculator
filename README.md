MegaCalc is a simple calculator application developed in Java using JavaFX for the graphical user interface (GUI). This application allows users to perform addition operations by entering two numbers and clicking on the "+" button. It provides a clean and intuitive interface for basic arithmetic calculations.
Features:
Addition operation: Users can input two numbers and add them together with the click of a button.
Clear function: The "Clear" button resets the input fields and clears the result display.
